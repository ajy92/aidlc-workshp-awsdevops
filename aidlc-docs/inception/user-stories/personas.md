# User Personas

## Persona 1: 김민수 (젊은 고객)
- **나이**: 28세
- **직업**: IT 회사 직원
- **디지털 친숙도**: 높음
- **특징**: 스마트폰 사용에 능숙, 빠른 주문을 선호, 새로운 기술에 거부감 없음
- **목표**: 종업원 기다리지 않고 빠르게 주문, 추가 주문도 즉시 처리
- **불만**: 피크 시간대 주문 대기, 종업원 호출 후 기다리는 시간

## Persona 2: 박영희 (중장년 고객)
- **나이**: 55세
- **직업**: 자영업자
- **디지털 친숙도**: 보통
- **특징**: 스마트폰 기본 기능은 사용 가능하나 복잡한 UI에 어려움, 큰 글씨와 직관적 버튼 선호
- **목표**: 메뉴를 천천히 살펴보고 주문, 주문 내역 확인
- **불만**: 작은 글씨, 복잡한 단계, 실수로 잘못 주문하는 것

## Persona 3: 이정호 (매장 운영자/관리자)
- **나이**: 42세
- **직업**: 한식당 사장
- **디지털 친숙도**: 보통
- **특징**: 매장 운영 경험 풍부, 효율적 운영에 관심, 매출 데이터 분석에 관심
- **목표**: 실시간 주문 확인, 메뉴 관리, 매출/고객 데이터 파악
- **불만**: 피크 시간 주문 누락, 수기 매출 정리, 인건비 부담
